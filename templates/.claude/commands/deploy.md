# /deploy コマンド

このコマンドを実行すると、デプロイ前チェックと安全なデプロイ手順を実行します。

## 実行手順

1. **デプロイ前チェック**:
   - 全テストが通過しているか確認 (`npm test` または `pytest`)
   - Lintエラーが0件であることを確認
   - `main` ブランチと差分がないことを確認
   - セキュリティスキャン実行

2. **環境確認**: デプロイ先の環境（dev/staging/prod）を確認

3. **デプロイ実行**:
   - 環境が `prod` の場合は必ずユーザーに確認を求める
   - デプロイログを記録

4. **デプロイ後確認**:
   - ヘルスチェックエンドポイントの疎通確認
   - 重要な機能の動作確認

## 安全制約

- **本番環境 (prod) へのデプロイは自動承認しない**
- デプロイ前に必ずバックアップが取れていることを確認
- ロールバック手順を事前に確認してからデプロイ
