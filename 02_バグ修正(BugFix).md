# 02 バグ修正（Bug Fix）

## Claude Code 起動コマンド

```bash
claude --dangerously-skip-permissions
```

## プロンプト指示

```
以下のバグを調査・修正してください。

【バグ概要】
- 発生場所: [FILE_PATH or MODULE]
- 症状: [症状の説明]
- 再現手順:
  1. [手順1]
  2. [手順2]
- エラーメッセージ（あれば）:
  ```
  [エラーメッセージをここに貼り付け]
  ```

【期待する動作】
[修正後に期待する正しい動作]

【実行してほしいこと】
1. 関連ファイルを調査して根本原因を特定する
2. 修正方針を Agent Teams（QA / 該当 Dev / Tester）で議論する
3. 最小限の変更で修正を実装する
4. 修正箇所に対するユニットテストを追加する
5. 既存テストがすべてパスすることを確認する
6. 再発防止策をコメントに記載する
7. git commit（fix: [バグ内容の要約]）する

【優先度】
[High / Medium / Low]
```

## 使用場面

- 本番バグの緊急修正
- テスト失敗の原因解消
- ユーザー報告の不具合対応

## ポイント

- エラーメッセージはできる限り完全なスタックトレースを貼り付ける
- 再現手順を明確にすることで AI の調査効率が大幅に上がる
